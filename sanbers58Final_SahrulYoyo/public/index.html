API Documentation

Products

Get All Products

GET

/products

Retrieve a list of all products.

Response

[
  {
    "id": "product_id",
    "name": "Product Name",
    "price": 100,
    "category": "Category Name",
    "qty": 50
  },
  ...
]

Example Request

GET /products

Create Product

POST

/products

Create a new product.

Parameters

Name Type Description Required
name string Name of the product Yes
price number Price of the product Yes
category string Category of the product Yes
qty number Quantity of the product Yes

Response

{
  "message": "Product created successfully",
  "data": {
    "id": "product_id",
    "name": "Product Name",
    "price": 100,
    "category": "Category Name",
    "qty": 50
  }
}

Example Request

POST /products
{
  "name": "Product Name",
  "price": 100,
  "category": "Category Name",
  "qty": 50
}

Get Product by ID

GET

/products/:id

Retrieve a single product by its ID.

Response

{
  "id": "product_id",
  "name": "Product Name",
  "price": 100,
  "category": "Category Name",
  "qty": 50
}

Example Request

GET /products/12345

Update Product

PUT

/products/:id

Update an existing product by its ID.

Parameters

Name Type Description Required
name string Name of the product No
price number Price of the product No
category string Category of the product No
qty number Quantity of the product No

Response

{
  "message": "Product updated successfully",
  "data": {
    "id": "product_id",
    "name": "Product Name",
    "price": 120,
    "category": "New Category Name",
    "qty": 40
  }
}

Example Request

PUT /products/12345
{
  "name": "Updated Product Name",
  "price": 120,
  "category": "New Category Name",
  "qty": 40
}

Authentication

Login

POST

/auth/login

Authenticate a user and retrieve a JWT token.

Parameters

Name Type Description Required
email string User's email Yes
password string User's password Yes

Response

{
  "token": "jwt_token",
  "user": {
    "id": "user_id",
    "email": "user@example.com",
    "role": "user"
  }
}

Example Request

POST /auth/login
{
  "email": "user@example.com",
  "password": "yourpassword"
}

Cart

Add Item to Cart

POST

/cart/add

Add an item to the cart.

Parameters

Name Type Description Required
product_id string ID of the product to add Yes
quantity number Quantity to add Yes

Response

{
  "message": "Item added to cart",
  "cart": {
    "items": [
      {
        "product_id": "product_id",
        "quantity": 2
      }
    ],
    "total": 200
  }
}

Example Request

POST /cart/add
{
  "product_id": "product_id",
  "quantity": 2
}

Remove Item from Cart

DELETE

/cart/:productId

Remove an item from the cart by product ID.

Response

{
  "message": "Item removed from cart",
  "cart": {
    "items": [],
    "total": 0
  }
}

Example Request

DELETE /cart/product_id
